// what is it
a calculator with a personality.

sentient mathematics is a lightweight desktop calculator with a retro crt aesthetic and a typing sound effect for every output. it handles all the standard stuff — arithmetic, trig, constants, exponents, variables, custom functions — but it also has opinions. type a big enough number and it'll complain. go past 1000 and it starts begging.

it started life as a simple terminal factorial finder. that legacy lives on whether you like it or not.

// how to use it
everything you can do.
arithmetic

standard math with + - * / and parentheses. works exactly how you'd expect.

exponents

use ^ for powers. e.g. 2^8 gives 256. works inside other functions too.

sqrt & log

sqrt(144) gives 12. log(1000) gives 3. log is base 10.

trig

sin() cos() tan() — all in degrees. e.g. sin(90) gives 1.

constants

built-in: pi π e tau phi. just type them anywhere in an expression.

variables

store values with x = 5, then use x in any expression. persists for the session.

custom functions

define with f(x) = x * 2, call with f(5). supports multiple params too.

ans

refers to the last result. e.g. do 5 * 5 then type ans + 1 to get 26.

debug mode

prefix any input with debug to get extra output info. use debugsilent to suppress the display.

⚠ the factorial thing

typing a bare whole number computes its factorial, not the number itself. so 5 gives Factorial of 5 = 120. this is not a bug. it's a historical artifact from when the whole program was just a factorial finder. numbers over 100 get roasted. numbers over 1000 get a full breakdown.
